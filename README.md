### Escrow_dapp

### Steps to Deploy Contract to Moonbase Aplha Testnet- Polkadot Parachain

cd backend
```javascript
1. truffle compile
//compile contract




2. truffle migrate run --network moonbase 
//deploy contract to testnet
```
